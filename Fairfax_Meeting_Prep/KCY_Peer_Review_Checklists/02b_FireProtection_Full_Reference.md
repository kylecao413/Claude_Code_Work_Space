### FS-XXX Life Safety Plan content (D10 expanded)
- All fire-rated walls delineated with type (fire barrier / partition / fire wall)
- Exit signs coordinated with electrical lighting plan

- Corridor + aisle dimensions
- Door + hardware schedule including special locking devices
- Boundary of all fire/smoke rated walls including atrium
- Travel distance + common path; egress to exit + public way
- Egress capacity calculations
- FEC locations
- Delayed-egress / special locking with symbols (card reader, motion sensor, push-to-exit)

## §F — Fire Protection / Life Safety (PRR §F — biggest section)

### F1. FM Commercial Plan Review Submittal Requirements compliance (4-19-2024 ed.) — entire document is required reading
This document distinguishes information required at building permit vs. that which may be deferred to shop drawing.

### F2. Fire site drawing approved (or latest copy attached) when scope includes:
- Any new building (incl. storage shed)
- Any addition
- Construction type decreasing in fire-resistance rating (i.e., increasing combustibility)
- New sprinkler installation
- Modifications to streetscape that decrease street access or alter fire lane
- Ground-floor pool addition
- Relocating existing FDC / underground fire line
- Changes to water supply / fire hydrant relocation

> **County will not approve building plan without approved/latest fire site drawing.** Use plan must match between site plan + building plan.

### F3. Emergency Lighting / Backup Power
1. Areas requiring emergency lighting + emergency power identified
2. Capacity of emergency systems for: egress lighting, exit signs, door locks, fire pumps, communication, elevators, other emergency equipment per code
3. Tank capacity, type, fuel, design # for tanks/generators identified + location dimensioned from buildings + property lines
4. Delayed egress utilized → emergency lighting at door clearly shown

### F4. Interior Finishes — combine with arch code analysis. Note compliance with §803, §804.

### F5. Special Use / Occupancy clearly identified (malls, atria, high-rise, hazardous areas, podium).

### F6. Special Processes / Materials / Hazards identified — elevators + Hazmat MAQ + control areas classified on floor plans.

### F7. Fire Protection Systems — required systems all identified
- Sprinkler per §903 (infrastructure only OR limited-area sprinkler design)
- Standpipe per §905
- Portable extinguishers per §906
- Fire alarm + detection per §907
- CO detection in I, R, E per §915

### F8. Deferred Shop Drawings — drawing notes shall state that the following will be submitted to FM Office:
- FP system: sprinklers, standpipes, alternative suppression, fire detection/alarm, fire pumps, kitchen hood
- Access control locks / special locking arrangements
- Tanks storing flammable + combustible liquids incl. generators

### F9. Smoke Control — three-step submittal process (see Smoke Control Supplemental Checklist)
1. **Rational Analysis Report** including special inspection plan + periodic test plan (§909.19, IFC 909.22.1) + integrated test plan (§901.6.2) — submitted **before** FM construction-plan approval. Reference Smoke Control Manual.
2. **Third-Party Inspector Qualifications** — resumes + AABC air-balancer certs — approved ≥ 30 days before SI begins.
3. **Final Test Report** (§909.18.8.3) — sealed by RDP. Required content:
   - Pressure differentials at doors per stair, per level, per atrium door (system in operation)
   - Pressure differentials across smoke barriers per smoke-control condition
   - How rational-analysis assumptions fit testing + environmental conditions
   - Verification of all sequences of operation (manual + automatic), panel functions, control + monitoring
   - Power transfer time + system performance
   - Visual verification of all rated walls/barriers/partitions/ceilings + penetration sealing; as-built submittal if changed
   - Detector activation + annunciation verified; min + max airflow tested
   - Damper operation per location + smoke-zone boundary
   - Air flow rates at exhaust + makeup vs. rational analysis
   - Drawings showing pressure measurement locations + times
   - Fan curve from manufacturer + fan operation during testing (V/A/RPM/belt tension)
   - Calibration certs for gauges/measurement devices ≤ 12 months
   - Identification of all devices (mfgr, nameplate, design value, measured value)
   - Deviations from rational analysis + limitations during testing
   - NFPA 4 integrated testing results (§901.6.2)
   - RDP certifies installation aligns with most current approved construction docs + rational analysis
   - Commissioning reports detailing steps to ensure design intent achieved

### F10. Smoke Control Record Number (Kyle's meeting takeaway #3)
**Record number must be provided prior to building plan approval.** Process:
- Create Smoke Control record per `Smoke Control PLUS Guidelines.pdf`
- Upon application approval, upload required documentation into PLUS | Fire and Rescue
- Cite the record number on plans
