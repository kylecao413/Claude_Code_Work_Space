# 06a — Mechanical Plan Peer Review Scan-Order Cheatsheet

**KCY Engineering Code Consulting LLC** · 2021 VMC · Fairfax County

---

## Round 1 — 5-min binary check

- [ ] Building Division coversheet provided
- [ ] Sealed by VA RDP / Class A Mechanical Contractor / Master HVAC tradesman
- [ ] Equipment schedule on plans with all air handling equipment
- [ ] Roof plan with all roof-top equipment (HVAC, exhaust fans, compressors, vent)
- [ ] Sequence of operations for AHE provided

---

## Round 2 — Sheet-by-sheet

### M0.0 — Cover + General (3 min)
- [ ] Code: 2021 VMC + amendments + ASHRAE 62.1 / 90.1 + IECC + manufacturer ICC-ES reports
- [ ] Type of construction + UL design # or prescriptive method for components based on required ratings
- [ ] Use of all rooms shown
- [ ] Roof plan w/ existing + new RTU equipment
- [ ] Equipment schedule for all AHE (fans, AC, furnaces, cooling towers, indoor/outdoor units)
- [ ] Sequence of operations for AHE
- [ ] Appliances listed + labeled per VMC §105
- [ ] Mechanical equipment installation per §304; guards per §304.11
- [ ] Condensate disposal per §307; secondary drain for indoor equipment per §307.2.3 + County Code §67.1-2-1 + Fairfax MS4 Permit

### M1.x — Ventilation (5 min)
- [ ] Every occupied space ventilated (natural or mechanical) per §402, §403
- [ ] Air intake opening per §401.4
- [ ] Recirculation per §403.2.1 + Table 403.3.1.1 footnotes
- [ ] **Min outdoor air calc table** per zone per Table 403.3.1.1 + Equations 4-1 to 4-8
- [ ] Occupant load + ventilation rate calc per Table 403.3.1.1
- [ ] OSA delivered to unit (cfm) ≥ OSA calculated (cfm)
- [ ] Design CFM for diffusers + grilles per space
- [ ] Air distribution > 2,000 cfm common supply/return → smoke detector in return upstream of OSA per §606.2.1; controls per §606.4

### M2.x — Exhaust (5 min)
- [ ] Exhaust outlet locations per §501.3.1 (Items 1–5)
- [ ] Exhaust + makeup + supply + return + relief maintain pressure equalization per §501.4
- [ ] Enclosed parking garage: 0.75 cfm/sf per §404.1; 0.05 cfm/sf with CO + NO2 detectors
- [ ] Stationary storage battery system spaces ventilated per §502.5
- [ ] Compressed medical gas storage (over IFC limit, no exterior wall): 1 cfm/sf exhaust through duct in 1-hr rated shaft per §502.9.1
- [ ] Repair garage / motor vehicle: source capture + 0.75 cfm/sf per §502.14
- [ ] Manicure/pedicure: exhaust ≤ 12" from chemical application per §502.20; ventilation per Table 403.3.1.1; operation per §502.20.1
- [ ] Toilet room exhaust 50/75 cfm per WC/urinal to exterior per §403.3.1.1 footnote e

### Clothes dryers (1 min — N/A unless laundry)
- [ ] Mfg installation instructions provided
- [ ] Plan details match mfg: duct size + location + termination
- [ ] Makeup air for installations > 200 cfm per §504.7
- [ ] Closet installations: opening ≥ 100 sq in OR makeup air
- [ ] Exhaust duct length max 35' or per mfg
- [ ] Duct does not extend into ducts/plenums

### Commercial Kitchen (5 min — N/A unless kitchen)
- [ ] Type II hood at all commercial food heat-processing equipment producing fumes/steam/odor/heat per §507.3
- [ ] Type I hood at all appliances producing grease vapors/smoke per §507.2
- [ ] UL or ICC-ES report for factory-built grease hood (UL 710)
- [ ] Exhaust + makeup air calcs for kitchen / entire space; all sources of exhaust + makeup air
- [ ] Type I hood — automatic fire suppression
- [ ] Type I hood exhaust fan interlock between appliance + hood per §507.1.1; mech makeup air auto-controlled to start with exhaust
- [ ] Type I hood + grease duct clearance per §507.2.6 + §506.3.6
- [ ] Type I hood ducts per §506.3.1 + enclosure §506.3.11
- [ ] Factory-built hood info: type/model/equipment schedule + exhaust + makeup CFM + grease filter type/area + UL 710 compliance + dimensions/gauge
- [ ] Duct grease accumulation + cleanout per §506.3.7 + §506.3.9
- [ ] Duct slope per §506.3.7
- [ ] Type I hood termination location per §506.3.13.1 (2)(3)
- [ ] Pollution control unit per §506.5.2

### M3.x — Combustion Air + Ducts + Dampers (4 min)
- [ ] Combustion + dilution air for fuel-burning appliances based on simultaneous operation per §701.1
- [ ] Corridors NOT served as supply/return/exhaust/relief/ventilation duct per §601.2 (exceptions apply)
- [ ] Return air in plenum per §601.2.1 conditions
- [ ] Ducts insulated per §604
- [ ] Return air not from closet/bathroom/toilet/kitchen/garage/boiler/furnace/unconditioned attic per §601.5
- [ ] HVAC component sizes (ducts, diffusers, registers, grilles) per §603.2
- [ ] Ductwork in rated floor/ceiling + roof/ceiling assemblies conform to listed assembly
- [ ] Ducts connecting multiple floors with penetration protection per §607.6.1 or §607.6.3

### Fire dampers + smoke dampers (3 min)
- [ ] Fire dampers where air distribution penetrates rated assemblies per §607.5
- [ ] Smoke dampers per §607
- [ ] Smoke detection for return air per §907 + §606.2

### Smoke control system (cross-ref FP) (2 min — if applicable)
- [ ] Connection to smoke removal system shown on mechanical drawings
- [ ] FP-side coordination — see `02b §F9-F10`

### Cross-discipline (2 min)
- [ ] Fire-rated walls + design # shown on mechanical plans
- [ ] Smoke detection for return air coordinated with FA
- [ ] Fire/smoke damper locations coordinated with arch + FP
- [ ] Fire duct wrap design assembly details
