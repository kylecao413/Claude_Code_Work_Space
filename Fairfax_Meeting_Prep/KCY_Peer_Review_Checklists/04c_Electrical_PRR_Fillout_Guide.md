## Fill-rules (recap)

| Mark | When | Comment |
|------|------|---------|
| ☑ Complies | Drawings show compliance | Blank or "OK" |
| ☐ N/A | Doesn't apply (e.g., no health-care occupancy) | One-line scope reason |
| ☐ + comment | Deficiency | "See Deficiency Log #X" |

No suggested correction language. No design opinions.

## §A — General Electrical Information (12 lines)
For each line, verify on drawings/schedules per `04b §A`. Critical lines:
- **Luminaire schedule with power ratings** — verify on lighting plan E1.x
- **Energy compliance method declared on coversheet** — verify Coversheet block
- **Riser diagram** — verify on E0.0
- **Panel schedules with V/A/phases/AIC/MCB-MLO** — verify each schedule
- **Drawings sealed by VA RDP OR Class A Contractor / Master Tradesman** — verify each sheet's title block
- **Electrical room doors with listed panic hardware opening in egress direction** — verify door schedule + hardware schedule + egress plan

## §B — Branch Circuits
- GFCI locations per §210.8: bathrooms, kitchens, rooftops, outdoors, sinks within 6', bathtubs/showers within 6'
- AFCI locations per §210.12: dwelling family / dining / living / parlor / library / den / sunroom / rec / closets / hallways
- Dwelling receptacles per §210.52 (commercial-only project → mark N/A "No dwelling units")
- Lighting outlets per §210.70

## §C — Service
For each line, verify on E2.x power plan + riser. If service is existing and not modified, the section may be partially N/A — note "Service existing, no modifications under this scope."

## §D — Feeder
Verify feeder schedules + tap-rule details on plans.

## §E — Grounding
Verify grounding detail on E0.0 with electrode + GEC + EGC sizes per code tables.

## §F — Overcurrent Protection
One line — verify on panel schedules + OCPD ratings.

## §G — Separately Derived Systems
Mark **N/A for whole section** if no transformers/generators in scope: "No separately derived systems in project scope."

## §H — Special Occupancies
Mark **N/A for whole section** unless H/I-2/Class I-III hazardous/place of assembly/garage.

## §I — Special Equipment
Mark each line:
- Signs: N/A unless tenant signage on Electrical scope
- Elevators: N/A unless elevator scope
- IT equipment: N/A unless data room/MDF
- Pools: N/A unless aquatic facility
- Fire pumps: N/A unless fire pump in scope

## §J — Emergency Systems
For each line, verify emergency power source + exit signs + egress lighting + battery duration. Storage battery ≥ 1.5 hours is hard requirement. **Sign at service equipment** indicating type + location of emergency power is often missed.

## §K — Legally Required Standby
Mark **N/A for whole section** if not required by code for occupancy type.

## §L — Miscellaneous
- Wiring in ducts/plenums per §300.22 — important if ceiling plenum return air
- Motor / appliance / elevator disconnects
- EMT — if EMT is the conduit method

## §M — Residential
Mark **N/A for whole section** for commercial-only: "Commercial project; no residential provisions apply."

## Comments column policy (Electrical-specific)

- ✅ "OK" or blank when Complies
- ✅ N/A justification at scope level — e.g., "No fire pump in scope" / "No transformers added"
- ✅ "See Deficiency Log #X" for deficiencies
- ❌ Suggested fixes
- ❌ "Designer should..."
