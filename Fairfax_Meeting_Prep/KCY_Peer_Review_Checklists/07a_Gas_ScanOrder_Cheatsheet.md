# 07a — Gas Plan Peer Review Scan-Order Cheatsheet

**KCY Engineering Code Consulting LLC** · 2021 VFGC + NFPA 99 (medical gas) · Fairfax County
> **Note:** Fairfax PRR combines Plumbing + Gas in §A–I. Plumbing covered in `05`. This cheatsheet covers §H (Fuel-Gas + Medical Gas).

---

## Round 1 — 3-min binary check

- [ ] Coversheet present
- [ ] Gas piping plan view + 2D riser diagram
- [ ] Sealed by VA RDP / Master Plumber / Master Gas Fitter
- [ ] Shutoff valve at each appliance
- [ ] Underground plastic piping has tracer wire OR approved equivalent

---

## Round 2 — Sheet-by-sheet (Gas only)

### G0.0 — Gas Plan + Riser (3 min)
- [ ] Code: 2021 VFGC + NFPA 99 (medical gas) + manufacturer specs
- [ ] Plan view of gas piping
- [ ] 2D riser diagram of gas piping
- [ ] Plumbing plan + gas riser coordinated
- [ ] Pipe sizing calculation provided
- [ ] Material schedule with manufacturer / model / specifications

### Fuel-Gas System (3 min)
- [ ] Piping materials per VFGC §402
- [ ] Pipe sizing per VFGC §402, §403 (capacity + pressure drop)
- [ ] Piping locations per VFGC §404
- [ ] Shutoff valve at each appliance per VFGC §409
- [ ] Underground plastic piping: tracer wire or approved equivalent per VFGC §404.17.3
- [ ] Pressure test requirements noted

### Medical Gas System (5 min — N/A unless healthcare)
- [ ] Pipe + fittings per NFPA 99 §5.1.10.1
- [ ] System design + construction per NFPA 99 §5.1.3.3.2
- [ ] Zone valve locations per NFPA 99 §5.1.4.6
- [ ] Shutoff valve labeling per NFPA 99 §5.1.11.2
- [ ] Source equipment + alarm panel locations
- [ ] Manifold + reserve calc

### Cross-discipline (1 min)
- [ ] Combustion air for fuel-burning appliances coordinated with mechanical (VMC §701.1)
- [ ] Tank/generator location dimensioned from buildings + property lines (FP cross-check `02b §F19`)
- [ ] Range hood interlock with gas shutoff (commercial kitchen — coordinate with mech)

---

## Round 3 — Output

For every FAIL → internal note → if true deficiency → client log row (Discipline = "Gas").
