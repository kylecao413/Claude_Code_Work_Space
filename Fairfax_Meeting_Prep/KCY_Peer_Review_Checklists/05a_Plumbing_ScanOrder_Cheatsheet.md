# 05a — Plumbing Plan Peer Review Scan-Order Cheatsheet

**KCY Engineering Code Consulting LLC** · 2021 VPC · Fairfax County
> **Note:** Fairfax PRR combines Plumbing + Gas in one form. Gas portion is covered by `07a/b/c`. This cheatsheet covers VPC items.

---

## Round 1 — 5-min binary check

- [ ] Coversheet present
- [ ] Sealed by VA RDP / Master Plumber
- [ ] Plan view + 2D riser diagram for water, storm, sanitary, gas
- [ ] Plumbing fixture schedule with removed/added/relocated fixtures
- [ ] Demolition plan (sanitary + water) for fixture-unit credits

---

## Round 2 — Sheet-by-sheet

### P0.0 — General + Cover (3 min)
- [ ] Code: 2021 VPC + 2021 IPC + ASME standards
- [ ] Coversheet
- [ ] Plumbing demolition plan for fixture-unit credits (existing tenant scope)
- [ ] Pipe sizes + stacks + risers + fixtures + equipment IDed
- [ ] Plan + riser diagrams coordinated with each other AND with site plan
- [ ] Plumbing details + equipment calcs
- [ ] Calc for **minimum number of required fixtures** per VPC §403.1

### County Ordinance compliance (1 min)
- [ ] Pools per County Code §69.1 (Health Department)
- [ ] Backflow per Fairfax Water Cross-Connection Control + Backflow Prevention Program Manual
- [ ] Separate sewer + water connections per County Code §65-6-8
- [ ] Car wash recycle systems per County Code §65-6-11

### P1.x — Plumbing Fixtures (4 min)
- [ ] Fixture schedule shows removed / added / relocated
- [ ] Min number of fixtures per VPC §403.1
- [ ] Drinking fountain / bottled water per VPC §410
- [ ] Separate facilities / multi-user gender-neutral / single-user / family or assisted-use toilet + bath per VPC §403.2 + §403.2.1
- [ ] Public + employee toilet facilities per §403.3
- [ ] Path to required fixtures per §403.3
- [ ] Fixture installation per VPC Ch 4

### P2.x — Sanitary / Vent System (5 min)
- [ ] Wastewater schedule
- [ ] Cleanout / manhole at building drain + sewer junction per VPC §708
- [ ] Cleanouts at change of direction per §708.1.4
- [ ] Pipe slope per §704.1
- [ ] Building sewer / drains / horizontal branches / offsets / stacks per §703 + §704
- [ ] Branch connections to offsets + stack offsets per §704.3, §711
- [ ] Fixture traps + trap seal protection per §1002.1, §1002.4
- [ ] Sanitary system vented per §910–§919, sized per §906
- [ ] Vent pipes + stacks per §905, §907, §908, §909
- [ ] Indirect waste + vacuum drainage per §802 + §715
- [ ] Sewage pump + vent per §712 + §906.5
- [ ] Waste discharge per §702.5, §702.6
- [ ] Backwater valves per §714

### P3.x — Water Supply System (4 min)
- [ ] Water service per §603.1 + §606
- [ ] Water distribution per §605.4
- [ ] Hot water + recirculation per §607
- [ ] Public hand washing + tempered water per §419.5 + §607.1.2
- [ ] Fixtures + equipment max flow + water consumption per §604
- [ ] Water heater thermal expansion control per §607.3
- [ ] Water heater safety pan + relief valves + discharge location per §504
- [ ] Water heater max temperature per §501.6
- [ ] Backflow protection per §608
- [ ] Backflow preventer schedule with type / mfg / model / location / standard

### P4.x — Storm + Non-Potable (3 min)
- [ ] Pipe slopes per §1106.3
- [ ] Horizontal + vertical storm system per §1106
- [ ] Cleanouts at change of direction per §708.1.4 + §1101.8
- [ ] Secondary roof drains per §1108
- [ ] Combined sewers per §1109
- [ ] Sump pumps / subsoil drains / building subdrains per §1111, §1112, §1113
- [ ] Greywater / rainwater / reclaimed non-potable per §1302, §1303, §1304

### P5.x — Interceptors + Separators (1 min)
- [ ] Location / sizing / venting / access per §1003 (grease, sand, oil, etc.)

### Accessibility (1 min)
- [ ] Water closet compartments + family/assisted-use per ANSI A117.1
- [ ] Plumbing fixtures accessible per ANSI A117.1

### Cross-discipline (2 min)
- [ ] Site plan + plumbing plan coordinated (sewer/water connection points match)
- [ ] Fire-rated walls shown w/ design # (FP cross-check from `02b §H`)
- [ ] Firestop/penetration details + design # (FP cross-check)
- [ ] Underground fire line size + location coordinated with sprinkler plan

---

## Round 3 — Output

For every FAIL → internal note → if true deficiency → client log row (Discipline = "Plumbing").

**Time budget:** ~30 min tenant interior / ~50 min new commercial.
