# 03c — Structural Fill-out Guide (Fairfax 2021)

**No standalone Structural PRR.** Structural items are filled into:
1. **Building PRR §J** (New Commercial form) — covered in `01c §J`
2. **Statement of Special Inspections** form `sip_ssi.pdf`
3. **Cold weather log** `sip-temperature-log.doc` (if winter pour)
4. **Stripping/shoring letter** `sip-stripping-letter.doc` (if applicable)
5. **Final Report of Special Inspections** `sip-final-report.doc` (post-construction, prior to CO)

**Pair with:** `03a` cheatsheet, `03b` reference, `00_KCY_Deficiency_Log_Template.docx`.

---

## Building PRR §J fill-rules (recap)

For each line in §J, mark Complies / N/A / Comment per `01c` rules. Specifically for Fairfax:

| §J line | Verify on plans | Common deficiency |
|---------|-----------------|-------------------|
| Structural design loads listed | S0.0 general notes table | Loads incomplete or generic non-Fairfax values |
| Live loads per Table 1607.1 | S0.0 list by occupancy | Missing or wrong factor |
| Roof snow Pg, Pf, Ce, I, Ct + Pd/W if applicable | **Pg = 25 psf for Fairfax** | Missing factor or non-Fairfax Pg |
| Wind + Tornado data | **V=90, Vult=115 for Fairfax** | Missing Vult, Risk Cat, GCpi, C&C |
| Seismic data | **Ss=0.129, S1=0.053 for Fairfax** | Missing SDS, SD1, SDC |
| Foundation plan, column + footing schedule, framing plans, connections, deep foundation full design | S1.x, S2.x sheets | Missing schedule, missing connection, deep foundation design incomplete |
| Calculations for special designs | Sealed calc package | Calcs not sealed, not dated |
| **SSI form complete + sealed by Structural Engineer** | `sip_ssi.pdf` attached | Missing pages, blanks on pages 2-3, wrong seal |
| ICC-ES eval reports for proprietary anchors / adhesives / fasteners | Drawing notes + attached reports | Out-of-date, wrong code year |
| Truss drawings + Truss Plan Coversheet | Submitted with permit OR deferred submittal note on drawings | No deferred-submittal note |
| Structural Observation if §1704.6.1 triggered | SE statement | Statement missing |

## Statement of Special Inspections (`sip_ssi.pdf`) — fill rules

### Page 1 — Project Information
- Project name + address + parcel + permit # — same as PRR
- Construction type + use group + occupancy
- Structural Engineer's name, license, contact
- **Sealed + signed + dated by Structural Engineer** (this is non-negotiable)
- Owner acknowledgment (signed by owner accepting SI program)

### Pages 2-3 — Special Inspection items
For **every line item** mark Yes (required) or No (not in scope). **No blanks.**

Common scope mappings:

| SI category | When to mark Yes |
|-------------|------------------|
| Soils (subgrade, fill, footings) | New foundation work |
| Concrete (cast-in-place / rebar / mix design / cylinders / mass / lightweight) | Any concrete work |
| Masonry (CMU, brick, grout) | Any masonry walls |
| Structural steel (welds, bolts, fabrication, erection) | Any steel framing |
| Cold-formed steel | Light-gauge framing in load-bearing or wind-bracing |
| Wood (engineered lumber, prefab trusses, glulam, LVL) | Wood framing > residential |
| EIFS | Any exterior insulation finish system |
| Post-installed anchors (mechanical, adhesive) | Any |
| Sprayed fire-resistant materials | When Type I/II requires sprayed |
| Smoke control systems | Any §909 system (cross-ref FP) |
| Storage racks > 12' | Any |
| Fire-rated assemblies | Any rated wall/floor/ceiling |
| Helical piles / aggregate piers / driven / caissons / augercast | Any deep foundation |
| Structural observation (§1704.6.1) | When Risk Cat III/IV or as required |

### Special Inspections Meeting
Note on cover sheet or general notes that "Pre-construction Special Inspections meeting is required prior to permit issuance."

## Cold weather concrete log (`sip-temperature-log.doc`)
If concrete pour in temps < 40°F:
- Daily ambient + concrete slab temperature recorded
- Heater type + duration logged
- Submitted to BO + on file with SI

## Stripping/shoring authorization (`sip-stripping-letter.doc`)
For cast-in-place concrete formwork removal / shoring removal / stressing:
- SE letter authorizing each phase
- Signed + dated by SE
- Submitted before stripping/stressing operation

## Final Report of Special Inspections (`sip-final-report.doc`)
- Compiled by Special Inspector / SE of record
- Lists every SI item from SSI + observation results
- All deficiencies + corrections logged
- Sealed + signed by SE
- Required for CO
