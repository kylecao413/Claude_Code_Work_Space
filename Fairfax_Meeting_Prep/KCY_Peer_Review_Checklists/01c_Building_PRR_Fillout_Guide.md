# 01c — Building PRR Fill-out Guide (Fairfax 2021)

**Form:** `Plan_Review_Records/2021/New_Commercial_Building/new-commercial-building-plan-review-record-2021.pdf` (or `Interior_Alterations/...` for tenant alterations).
**Purpose:** This is the **deliverable to Fairfax County** — the official document the County keeps on file. KCY peer reviewer fills it out and submits with the construction documents.
**Pair with:** `01a` cheatsheet (what to look at) + `01b` reference (code language) + `00_KCY_Deficiency_Log_Template.docx` (client-facing report).

---

## How to fill each section

For **every line item** the PRR has 3 possible marks:

| Mark | When to use | What to write in Comments column |
|------|-------------|----------------------------------|
| ☑ Complies | Drawings clearly show compliance | (leave blank, or "OK") |
| ☐ N/A | Item does not apply to this project | One-line justification — *e.g.* "No mezzanines proposed" / "Single-tenant building" |
| ☐ (uncheck both) + Comment | Deficiency found | "See Deficiency Log #X" — log row has the observation + code |

**Iron rule:** Do NOT write suggested corrections in the PRR Comments column. The County's PRR has a "Comments" column to note status; it is not where peer reviewer prescribes design fixes. Same conflict-of-interest reason as the deficiency log.

## §A — Administrative Requirements *(New Commercial: §B; Interior Alt: "Administrative Requirements" block)*

| PRR line | Cheatsheet ref | What "Complies" looks like | What "N/A" looks like |
|----------|----------------|----------------------------|------------------------|
| Building Plan Review Coversheet attached or embedded | Round 2 / A0.0 | 11x17 coversheet OR embedded sheet, all blocks filled | (Never N/A — always required.) |
| Clear scope of work statement on plans | Round 2 / G0.x | Narrative paragraph identifying use, type, scope | (Never N/A.) |
| Statement of Special Inspections + signed by SE | Round 2 / S-sheets | SSI form complete, structural engineer's seal/signature | "No structural work" — only valid for pure interior reno with no structural mods |
| Drawings reference all applicable codes | Round 2 / G0.x | List of 2021 VCC + ICC + amendments + ANSI A117.1-2009 + IECC | (Never N/A.) |
| ICC-ES eval report for EIFS / proprietary | Round 2 / Submittals | Report copy attached + report # called out on drawing | "No EIFS or proprietary devices" |
| Industrialized Building Affidavit | Round 2 / Submittals | County `indust.pdf` form complete | "Not an industrialized building" |
| Designer contact info on drawings | Round 1 | Title block on each sheet | (Never N/A.) |
| Drawings/details/calcs sealed by VA RDP | Round 1 | Each sheet sealed; calc package sealed | (Never N/A for commercial.) |

> **Interior Alterations adds:** Key plan if multi-space; alteration level classified per VEBC §103.9 + §601.2; location of alterations noted; repairs identified per VEBC Ch 5; building-related proffer/condition + compliance method; designer name/address/occupation OR sealed drawings.

## §G — Means of Egress *(highest deficiency volume — fill carefully)*

For each line, if Complies, leave Comments blank. If a deficiency, write `See Deficiency Log #X`.

PRR lines map to VCC sections:
- Occupant load (§1004.5) — furniture plan supports OL
- Number/configuration of exits (§1006, §1007)
- Egress illumination (§1008)
- Accessible MoE (§1009)
- Doors/gates/turnstiles (§1010) — door schedule + hardware schedule
- Stairways (§1011), Ramps (§1012), Exit signs (§1013), Handrails (§1014), Guards (§1015)
- Egress access — intervening, common path, travel distance, aisles (§1016, §1017, §1018)
- Corridors (§1020) — width, dead-end, continuity
- Interior exit stairways/ramps (§1023), Exit passageways (§1024), Horizontal exits (§1026), Exit discharge (§1028)

---

## §H — Accessibility

PRR lines map to VCC §1104 (route), §1105 (entrance), §1106 (parking), §1107 (dwelling/sleeping units + Type B), §1108.2 (assembly seating), §1109 (fixtures, drinking fountains), and ANSI §404 (door approaches), §405 (ramps), §902 (counters), §602 (drinking fountains), Ch 6 (plumbing fixtures).

- Type B dwelling units → only applies to multi-family residential. Mark N/A on commercial-only with note.
- Coversheet accessibility radio (Fully Compliant / Technically Infeasible / Upgrades Required) must match what the drawings demonstrate. If "Upgrades Required", verify cost documentation matches actual scope per VEBC Ch 4.

---

## §J — Structural

For each line: verify the data/document is on plans; mark Complies if present, deficiency if missing or wrong. Fairfax-specific values:

- Pg = 25 psf
- V = 90 mph (40 m/s) basic; Vult = 115 mph (51 m/s)
- Ss = 0.129, S1 = 0.053

If the drawings list values that DO NOT match these (and they're for Fairfax), it's a deficiency.

---

## Comments column policy (KCY house rule)

- ✅ "OK" or blank when Complies
- ✅ One-line reason when N/A — must be true at scope level (not just "designer didn't draw it")
- ✅ "See Deficiency Log #X" when found non-compliant
- ❌ Suggested fixes
- ❌ Plan-by-plan opinions
- ❌ Internal cross-references like "PRR §F1 line 3"
