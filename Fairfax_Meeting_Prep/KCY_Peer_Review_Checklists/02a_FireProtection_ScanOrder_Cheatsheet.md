# 02a — Fire Protection Plan Peer Review Scan-Order Cheatsheet

**KCY Engineering Code Consulting LLC** · 2021 VCC · Fairfax County Office of the Fire Marshal (OFM)
Pair with `02b` (full reference), `02c` (PRR fill-out), `00_KCY_Deficiency_Log_Template.docx`.

---

## Round 1 — 10-min binary check

- [ ] **Smoke Control Record number** assigned + cited on plans (if smoke control system applies) — Kyle's meeting takeaway #3
- [ ] Fire site drawing approved by OFM (or latest copy attached) for: any new building, addition, decreased construction-type rating, new sprinkler install, FDC/UG fire line relocation, fire-hydrant/water-supply changes, ground-floor pool addition, streetscape changes affecting fire lane
- [ ] Door schedule + door hardware schedule on plans (specific hardware sets, NOT "Existing to Remain", NOT "Refer to Specs")
- [ ] Fire-rated walls clearly delineated on **Architectural + MEP + Fire + IT** plans (not just Arch)
- [ ] Coversheet present + complete; designer info per discipline; key plan + true north on each sheet
- [ ] Plans labeled "For Construction" — NOT "For Bid Only", "Permit Only", "<100% Progress"
- [ ] Min scale 1/8"=1' (commercial) + 1/4"=1' for details + graphic scale present

---

## Round 2 — Sheet-by-sheet

### Coversheet + Code Analysis (3 min)
- [ ] Code editions stated incl. NFPA standards + edition (e.g., NFPA 13-19, NFPA 72-19, NFPA 80, NFPA 4)
- [ ] VEBC questions answered (purpose, repairs, alterations, level, change of occupancy, historic, moved)
- [ ] Use group, type of construction, height, area, sprinkler type (13/13R/13D), monitored Y/N, high-rise Y/N

### Fire/Life Safety Plan FS-XXX (10 min — biggest deficiency volume)
- [ ] Sheet exists per floor + named FS-XXX (FS-101 = ground floor convention)
- [ ] All fire-rated walls delineated with type (fire barrier / partition / fire wall) + UL # or VCC Ch 7 reference
- [ ] Smoke barriers shown including atrium boundary
- [ ] Exit signs coordinated with electrical light plan
- [ ] Exits + horizontal exits with OL supported by each exit
- [ ] Travel distance + common path of travel — show calc + allowable + actual
- [ ] Egress capacity of each exit door + stair (exit width vs OL × factor)
- [ ] Door + hardware schedule including special locking devices
- [ ] OL calculated per Table 1004.5 by function; both design OL and furniture-based OL where applicable, using greater
- [ ] Total OL per floor + per building
- [ ] Remoteness between required exits (1/2 diagonal, 1/3 sprinklered) clearly shown
- [ ] FEC (fire extinguisher cabinet) locations (max 75' travel)
- [ ] Delayed egress / special locking shown with symbols (card reader, motion sensor, push-to-exit)
- [ ] Locations of access controls / security locking IDed within egress paths
- [ ] Atriums + horizontal exit boundaries clearly identified
- [ ] Furniture plans with seating configuration
- [ ] Range hood pull station: 10' min – 20' max from hood, near means of egress
- [ ] Refuge area (I-2) calc + dimensions

### Construction Type matrix (3 min)
- [ ] Matrix on plans listing per-element rating: primary structural frame / bearing wall (ext + int) / non-bearing (ext + int) / floor / roof / shaft walls / exit stairs / exit passageway / elevator shaft + lobby / construction barriers / mixed-use separation / podium separation / horizontal exit / fire wall — each with UL # or Ch 7 ref
- [ ] Podium / multi-building → matrix per building
- [ ] Existing-building construction type verified
- [ ] FD-XXX sheets for rated assembly details (head of wall, curtain wall, special doors)

### Site Drawing (2 min)
- [ ] Approved (or latest) FM site drawing in plan set
- [ ] FDC primary on address side; secondary FDC for high-rise remote
- [ ] PIV location (post-indicator valve) if present
- [ ] Underground fire-line size, type, material on plumbing or sprinkler plan
- [ ] Fire hydrants + water-supply data per PFM Ch 9
- [ ] Fire lanes + work/laydown/staging areas don't impede FD access or egress
- [ ] Address on site plan = construction doc address = permit application address
- [ ] Construction type + use group on site plan = building plan

### Fire Protection systems (5 min)
- [ ] Sprinkler system per §903 — NFPA 13/13R/13D type stated
- [ ] Standpipe per §905 — Class I/II/III + locations
- [ ] Fire hose valves at horizontal exits, exit stairs, intermediate landings (visible from top), exit passageways, and roof/highest landing of stair with roof access
- [ ] Portable fire extinguishers per §906 — locations + max 75' travel
- [ ] Class K-rated extinguisher for cooking equipment (solid fuels, vegetable/animal oils/fats)
- [ ] Fire alarm + detection per §907 — manual pull station ≤ 5' from exit door, riser schematic, DACT for central station, FACP per Code Reference Package
- [ ] Audible water-flow alarm (exterior, near sprinkler room or FDC)
- [ ] CO detection in Group I, R, E per §915
- [ ] Smoke control per §909 — record number cited on plans; rational analysis report submitted; SI plan + periodic test plan + integrated test plan attached
- [ ] Mini-horns / strobes / speakers per §907.5.2 + NFPA 72-19 §18.5.5
- [ ] In-building emergency communications coverage per §918 (high-rise wired phones per §907.2.13.2)
- [ ] Smoke and fire damper locations on Mechanical + duct rating notes
- [ ] Smoke detection for return air per §907 + VMC §606
- [ ] Hazmat: control areas + MAQ + classification (combustible liquid, flammable solid, etc.)
- [ ] High-pile storage: declared + acknowledgment letter if "could accommodate" but not designed for it
- [ ] Tank capacity, type, fuel, design number for any flammable/combustible tank or generator + dimensioned from property + structures

### Doors + Hardware (3 min)
- [ ] Min clear width 32" (41.5" in I-2; max swinging leaf 48")
- [ ] All hardware sets explicitly defined per door — no "Existing to Remain", no "Refer to Specs"
- [ ] Card reader / access-control door — code section cited
- [ ] Two-way communication device where required
- [ ] Egress illumination on EXTERIOR side of exit door

### Phased Construction (if applicable, 2 min)
- [ ] Phasing plan with: scope per phase, dates/duration, temporary walls, path of egress per phase, emergency lighting + exit signs per phase
- [ ] Number of permit applications per phase
- [ ] Submittals complete per phase; "for reference only" markers on supporting docs from other permits

### MEP + Telecom cross-checks (3 min)
- [ ] Mechanical: rated walls shown with design # + smoke detection for return air + dampers + duct wrap detail
- [ ] Plumbing: rated walls + room/grid match arch + firestop/penetration design # 
- [ ] Electrical: exit signs to two distinct exits when 2 required (§1013) + emergency/night lighting + illumination at exits AND throughout exit discharge to public way (§1008) + rated walls
- [ ] Telecom: rated walls + firestop/penetration design #

### Smoke Control specific (if applicable) — Submittal step gate
- [ ] **Step 1:** Rational analysis report + SI plan + periodic test plan (§909.19, IFC 909.22.1) + integrated test plan (§901.6.2) submitted **before FM construction-plan approval**
- [ ] **Step 2:** Third-party inspector qualifications (resumes + AABC certs) approved ≥ 30 days before SI
- [ ] **Step 3:** Final test report — sealed by RDP, includes pressure differentials, sequence of operations, damper/airflow data, calibration certs ≤ 12 months, NFPA 4 integrated test results
- [ ] Reference: 2021-SIP Ch 16 + Smoke Control Manual + Supplemental Checklist

### Deferred Shop Drawings (note check)
- [ ] Drawing notes state shop drawings will be submitted to FM Office for: sprinklers, standpipes, alternative suppression, fire detection/alarm, fire pumps, kitchen hood, access-control locks, flammable/combustible tanks, generators

### Revisions (resubmittal only)
- [ ] Revised sheets marked on Drawing Index
- [ ] Cover letter listing all changes + sheet numbers
- [ ] Revision clouds + delta + revision number matching block + date

---

## Round 3 — Output

For every FAIL, internal note → if true deficiency → client log row (Discipline = "Fire Protection"). No suggested fixes.

**Time budget for typical commercial FP review:** ~50 min single-floor / ~90 min multi-floor or smoke control involved.
