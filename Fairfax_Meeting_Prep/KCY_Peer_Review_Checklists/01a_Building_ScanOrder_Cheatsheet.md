# 01a — Building Scan-Order Cheatsheet

**KCY Engineering Code Consulting LLC** · 2021 VCC / VEBC · Fairfax County
Use this for the actual review. Open the full reference (`01b`) only when you need code language. Tick each line; flag any FAIL into internal notes → escalate to client deficiency log.

---

## Round 1 — 10-minute binary check (kill-switch items)

If any of these fail, stop and ask designer to fix before doing the deep review:

- [ ] Coversheet present (11x17 or embedded), all blocks filled
- [ ] Door schedule + door hardware schedule on plans
- [ ] SSI form attached, signed/sealed by Structural Engineer, all yes/no marked
- [ ] Designer of record + license # + seal + date on each sheet (or coversheet w/ TOC)
- [ ] Code edition stated (2021 VCC or 2021 VEBC + which alteration level)

---

## Round 2 — Sheet-by-sheet

### A0.0 — Coversheet (5 min)
- [ ] Project name, address, floor/suite, work description
- [ ] Code edition + alteration level (Repair / L1 / L2 / L3 / Change of Occupancy)
- [ ] Occupancy/Group(s), Type of Construction, Stories
- [ ] High Rise / Critical Structure / Unlimited Area / Atrium boxes
- [ ] Sprinkler (full/partial/none), Fire Alarm, Central Station, Standpipes
- [ ] Hazmat boxes (combustible / flammable / high-pile)
- [ ] Fire-resistance rating cells: floor/ceiling, roof, columns, beams, corridors, tenant walls
- [ ] Accessibility radio: Fully Compliant / Technically Infeasible / Upgrades Required + cost numbers (cost of alterations + cost of upgrades + items list)
- [ ] Energy compliance: ASHRAE 90.1 OR VECC
- [ ] Designer info per discipline (B/S/M/E/P): name, phone, email, license #
- [ ] Zoning case # + Code Compliance case # (if any)
- [ ] Building-related proffers listed with drawing reference
- [ ] Tenant block (if alteration): purpose, gross area/floor, area of work, gross tenant area
- [ ] Base building design code/year stated (existing buildings)

### G0.x / Code Summary sheet (3 min)
- [ ] Applicable codes + amendments listed
- [ ] Allowable vs actual height (ft + stories) shown
- [ ] Allowable vs actual area per story shown; frontage increase math if used
- [ ] Mixed-use type + separation per VCC §508
- [ ] VEBC compliance method (existing only)
- [ ] Site plan reference + civil set agreement on footprint, use group, type

### A0.x / Life Safety Plan(s) (10 min — biggest rejection source)
- [ ] Occupant load by space, supported by furniture plan (Table 1004.5)
- [ ] Number of exits + exit access doorways adequate (Table 1006.2.1, 1006.3.2)
- [ ] Common path of egress within Table 1006.2.1
- [ ] Travel distance within §1017
- [ ] Exit remoteness: 1/2 diagonal, or 1/3 if sprinklered
- [ ] Corridor width ≥ Table 1020.2; dead-ends within §1020.4
- [ ] Corridor continuity to exit + fire-resistance rating per §1020
- [ ] Corridor doors don't reduce egress to < 1/2 required width
- [ ] Open door projection ≤ 7" into egress width
- [ ] Means of egress illumination shown including at exit discharge (§1008)
- [ ] Exit signs at exits + along path (§1013)
- [ ] Accessible means of egress + areas of refuge if required (§1009)
- [ ] Egress through intervening spaces — only one allowed; not through kitchens/storage etc. (§1016)

### A1.x / Architectural — Floor Plans (8 min)
- [ ] Fire walls, fire barriers, fire partitions tagged with rating + UL # (§706–708)
- [ ] Vertical openings protected (§712); shaft enclosures (§713)
- [ ] Opening protectives in rated walls — door rating + glazing rating (§716, §717)
- [ ] Concealed-space fireblocking + draftstopping (§718)
- [ ] Mezzanines: area ≤ 1/3 room (§505.2.1), egress (§505.2.2), openness (§505.2.3)
- [ ] Equipment platforms: definition (§505.3.1), sprinkler (§505.3.2)
- [ ] Tenant separation rated walls if required
- [ ] Clear scope-of-work statement

### A2.x / Door Schedule + Hardware Schedule (5 min)
- [ ] Door schedule lists every door with type, size, rating, hardware set ref
- [ ] Hardware schedule shows function: closer, latch, panic, fire-exit, sensor-release, delayed-egress, electric strike
- [ ] Manual flush bolts NOT on egress doors (§1010.1.9.4)
- [ ] Auto flush bolts only on inactive leaf of pair (§1010.1.9.3(3))
- [ ] Key-operated lock on main exterior B/F/M/S — meets all 3 of §1010.1.9.3(2)
- [ ] Clear width ≥ 32" at 90° (§1010.1.1)
- [ ] Side-hinged + swing in egress direction where OL ≥ 50 (§1010.1.2.1)
- [ ] Hardware operable w/o tight grasp/pinch, 34"–48" AFF (§1010.1.9.1, §1010.1.9.2)
- [ ] Unlatching ≤ 1 operation (§1010.1.9.5)
- [ ] Panic/fire-exit hardware: A or E w/ OL ≥ 50, H, electrical room ≥ 1200 A (§1010.1.10)
- [ ] Fire doors self-closing + latching (§716.5.9, NFPA 80)
- [ ] Sensor-release: card reader + motion + manual unlock < 5'; tied to FA; ≤ 2 per egress path (§1010.1.9.8)
- [ ] Delayed-egress: sprinklered (except A/E/H); ≤ 1 per egress path; 6 items per §1010.1.9.7
- [ ] Stairway doors lockable only if §1010.1.9.11 exception met
- [ ] Doors-in-series: 48" between (§1010.1.8); landings ≥ 44" depth ≥ door width (§1010.1.5–.6)

### A3.x / Restrooms + Accessibility (5 min)
- [ ] Accessible route does NOT pass through kitchens/storage/closets (§1104.5)
- [ ] ≥ 60% public entrances accessible (§1105.1)
- [ ] Door approaches: 18" pull / 12" push w/ closer+latch (ANSI 404.2.3.2)
- [ ] All required restroom fixtures accessible (§1109.2–§1109.5, ANSI Ch 6)
- [ ] 60"x59" clear at WC; sink does not overlap (ANSI 604.3)
- [ ] WC compartment doors comply (ANSI 604.9.3)
- [ ] Ambulatory WC where 6+ WCs/urinals in single restroom (§1109.2.2)
- [ ] Family/assisted-use restroom in M/A where 6+ aggregate WCs (§1109.2.1)
- [ ] Drinking fountain: high/low or 2 sep (§1109.5)
- [ ] Counter heights, seating, dining, bar — all 5% accessible (§1108.2, §1109.11)
- [ ] Type B units indicated, count per Table 1107.6.1.1 (§1107.6.1)
- [ ] If "Technically Infeasible" or "Upgrades Required" on coversheet — costs documented (VEBC Ch 4)

### A4.x / Stairs + Ramps + Handrails + Guards (3 min)
- [ ] Stairway width, headroom, riser/tread compliant (§1011)
- [ ] Ramp slope 1:12 max, edge protection, handrails both sides if rise > 6" (§1012)
- [ ] Handrails per §1014 (height, extension, graspability)
- [ ] Guards per §1015 — required where surface > 30" AFG; opening size limits

### A5.x / Interior Finishes (1 min)
- [ ] Class A/B/C interior finishes per Table 803 + §804 by space
- [ ] Carbon monoxide detection in Group I/R/E (§915)

### S-sheets / Structural (5 min — peer review only; FFX does official)
- [ ] Design loads listed: live, snow, wind, seismic (per §1603)
- [ ] **Fairfax values:** Pg=25 psf, V=90 mph (Vult=115), Ss=0.129, S1=0.053
- [ ] Live load table per Table 1607.1 by occupancy
- [ ] Snow: Pg, Pf, Ce, I, Ct + drift Pd/W if applicable
- [ ] Wind: V, Vt, Vasd, risk cat, exposure, GCpi, C&C pressure
- [ ] Seismic: risk cat, IE, mapped + design SRP, site class, design cat, SFRS, base shear, Cs, R, analysis procedure
- [ ] Foundation plan, column/footing schedule, framing plan
- [ ] Connection details
- [ ] Allowable bearing pressure on plans matches geotech
- [ ] Geotech report sealed by VA RDP if soil > 2,000 psf or any deep foundation
- [ ] Deep foundation full design if used (§1810)
- [ ] Special foundations (helical/aggregate piers) — eval report + design + special inspection
- [ ] Statement of Special Inspections, signed/sealed by SE, all items marked
- [ ] ICC-ES eval reports for proprietary anchors/adhesives/fasteners
- [ ] Truss Plan Coversheet + truss drawings (or deferred-submittal note)
- [ ] HVAC equipment attachment detail + calc

### Energy / Envelope (2 min)
- [ ] Energy compliance method on coversheet matches drawings (ASHRAE 90.1 or VECC)
- [ ] R-values match VECC C402.1.3 (mass / wood / metal / roof / attic)
- [ ] Vestibule on every non-exempt entrance, self-closing doors, doors-in-series clearance (VECC C402.5.7)

### Submittals & Forms (final pass)
- [ ] ICC-ES report for EIFS + other proprietary materials attached (§109.3, §1408)
- [ ] Industrialized Building Affidavit if applicable
- [ ] Truss Plan Coversheet + drawings if available
- [ ] Building-related proffers incorporated; drawing references match coversheet
- [ ] Resubmittal letter addressing each prior comment (resubmittals only)
- [ ] Key plan for multi-tenant / large floorplate
- [ ] Separate-permit notes for retaining walls, sheeting/shoring, modular furniture, future tenants

## Round 3 — Output

For every FAIL above:
1. Open `<Project>_Internal_Notes.md` — add row with PRR §, sheet, observation, code, County rejection-language ref.
2. Decide if it's a true deficiency or just clarification. If deficiency:
3. Open `00_KCY_Deficiency_Log_Template.docx` — add row with Discipline / Sheet / Observation + Code citation. **No suggested correction.**
4. After review complete, fill Fairfax PRR form using `01c` guide; reference deficiency log row numbers in Comments column where needed.
