# 09 — Other Agency Reviews (Coordination Pointers)

**KCY Engineering Code Consulting LLC** · Fairfax County
**Scope:** Lightweight pointer document. Fairfax LDS coordinates with multiple sister agencies. As peer reviewer we don't perform their reviews, but we **flag for the client** when the scope of work triggers an agency review they need to plan for.

This is **not a checklist we use to review plans** — it's a "when to tell the client they have another approval to chase" reference.

---

## Health Department (Fairfax HD)

**Trigger when project scope includes:**
- Pools, spas, splash pads
- Food service (restaurant, commercial kitchen, food market)
- On-site wastewater (septic) — non-municipal sewer
- Wells (potable water)
- Body art / tattoo / piercing facilities

**Flag for client:** "Health Department review required. Coordinate with `https://www.fairfaxcounty.gov/hd/eh` (Environmental Health). Plan for parallel approval timeline."

---

## Site / Site Development

**Trigger when project scope includes:**
- Any new building or addition with grading > a threshold area
- New impervious surface
- Modifications to streetscape, fire lane, sidewalk, curb cut
- Stormwater management changes

**Flag for client:** "Site Development review required. Coordinate with LDS Site Development Division. Site plan must be approved before Fairfax FM Office approves fire site drawing for building (see `02b §F2`)."

---

## Wastewater Review

**Trigger when project scope includes:**
- Connection to public sewer (capacity check)
- Industrial or special discharge (food, lab, dental, manufacturing)
- Backflow / cross-connection (cross-ref `05b §B`)

**Flag for client:** "Wastewater review required for connection capacity + special discharge. Coordinate with `https://www.fairfaxcounty.gov/landdevelopment/wastewater-review-building-permits`."

---

## Zoning

**Trigger when project scope includes:**
- New use (any change of occupancy)
- Site expansion / addition
- Variance / special exception from zoning ordinance
- Proffer / development condition compliance

**Flag for client:** "Zoning compliance review required. Building-related proffers must be incorporated into building drawings (cross-ref `01b §A14` + Coversheet block). Building plan will not be approved unless zoning matters resolved. Coordinate with Department of Planning + Zoning."

---

## Fire Marshal (Commercial)

**Trigger:** All commercial projects coordinate with the Office of the Fire Marshal (OFM). The Fire Plan Review (`02`) is the primary venue, but separate FM-specific items exist:
- Fire site drawing approval (cross-ref `02b §F2` — required for new buildings, additions, decreased construction-type rating, new sprinkler installation, FDC/UG fire line / hydrant relocation, ground-floor pool, streetscape changes affecting fire lane)
- Smoke control record (cross-ref `02b §F9-F10`)
- Hazmat permits (over MAQ)
- High-pile combustible storage acknowledgment letter (cross-ref `02b County rejection-language`)
- Fireworks / pyrotechnics
- Outdoor cooking / commercial kitchen approvals
- Fire pump acceptance testing

**Flag for client:** "Several FM-specific approvals required outside the building plan review track. See `Other_Agency_Reviews/Fire_Marshal_Commercial/fire_marshal_commercial_plan_review_submittal_requirements.pdf` for full submittal requirements. Some items (smoke control, fire pump, kitchen hood) can be deferred shop drawings — note on plans."

---

## Coordination matrix (when one agency blocks another)

| Trigger | Agency that must approve first | Building plan blocked? |
|---------|-------------------------------|-------------------------|
| Site plan changes | Site Development → FM | Yes — fire site drawing needs site plan; building plan needs both |
| New sewer connection | Wastewater | Yes — building plan blocked |
| Change of occupancy | Zoning | Yes — building plan blocked |
| Smoke control system | OFM Smoke Control record (`02b §F10`) | Yes — record # required prior to building plan approval |
| Restaurant + commercial kitchen | Health Dept + OFM (hood / suppression) | Often yes |

---

## Output

- This document is **NOT** part of the deficiency log.
- When a client coordination need is identified, write a **separate "Coordination Notes" section** in the project's KCY peer review report (cover letter or separate page after deficiency log).
- Use language like: *"Note: Project scope appears to trigger a [Agency] review. Recommend the client confirm parallel approval is in progress before building plan submission."* — this is informational, not a deficiency.
