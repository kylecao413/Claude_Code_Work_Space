# Copilot Quick-Prompt Card — Fairfax Training

**How to use:** Keep this open in a side window. Copy any line, paste into Copilot chat, fill in the `[brackets]`, hit Enter. Most return in 5–10 seconds.

> ⚠️ Microsoft 365 Copilot can only see what's in the meeting **transcript**. If Fairfax doesn't enable transcription, Copilot won't "hear" the conversation — it will only work on text **you paste in**. Plan accordingly: keep a notepad to type 1-line summaries every few minutes that you can drop into Copilot if needed.

## Setup tips before joining

1. **Pin Copilot panel** in Teams (right rail). Keep it open the whole meeting.
2. **First message every meeting**: paste `Fairfax_Copilot_Feed_Prompt.md` Version A. Wait for "Loaded — ready" before the meeting actually starts.
3. **Recording + Transcription**: only the meeting **organizer** can start them. If Fairfax doesn't, Copilot can't auto-listen — fall back to typing notes for it.
4. **Live Captions translation**: you can turn this on yourself anytime. `More → Language and speech → Turn on live captions → Settings → Spoken English / Caption Chinese (Simplified)`.
5. **Copy this card**: keep `Fairfax_Copilot_Quick_Prompts.md` open in a second window. Don't try to remember prompts — copy/paste is faster.
