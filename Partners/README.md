# Partners

Curated roster of architect / GC / designer / partner contacts BCC works with directly.

**Distinct from `BCC_Outreach_Tracker.md`** (cold-outreach pipeline state — sent / followup / replied / won / lost).
This folder is for people Kyle has met, partnered with, or been referred to — the kind of contacts you'd flip to when deciding "who should I send the residential pricing sheet to" or "who do I know at firm X".

## Convention

- **One markdown file per company.** Filename = company name with underscores.
- **Each file = one table** with columns: Name | Role | Phone | Email | Added (YYYY-MM-DD) | Notes
- **Header line** = full company name and any sub-tagline ("Building & Design Construction", "Architects", etc.).
- When Kyle hands over a new contact:
  - If the company file exists → append a row.
  - If not → create a new file `<Company_Name>.md` using the existing files as the template.
- Don't delete contacts when they leave a firm — move the row to a `## Past contacts` section at the bottom of the file with the departure note.

## Current files

Run `ls Partners/` for the up-to-date list.
