# BuildingConnected 提案模板

**优先使用**：脚本会优先读取 Building Code Consulting 根目录下的你的模板：
- `DC Code Compliance Proposal Template.docx`（第三方检测 / Code Compliance）
- `DC Plan Review Proposal Template.docx`（Plan Review）

若上述路径不存在，再使用本目录下的同名文件。

模板中可使用占位符（将自动替换）：

- `{{Client}}` – 客户名称  
- `{{Project}}` / `{{ProjectName}}` – 项目名称  
- `{{PricePerVisit}}` – 单价 $/visit  
- `{{EstVisits}}` – 预估检测次数  
- `{{Total}}` – 合计金额  
- `{{ScopeNotes}}` – 范围说明/项目描述摘要  
- `{{BuildingCodeConsulting}}` – Building Code Consulting LLC  
- `{{DC Third-Party}}` – DC Third-Party agency  
- `{{Yue Cao}}` – Yue Cao (PE, MCP)  

参考已有提案风格：RIVIAN Flagship、2229 M ST NE、JW Marriott、Winmar、Joe & Juice、1812 H PL NE 等（见 Projects 下各客户文件夹）。
