# South Carolina — SC LLR Board of Engineers & Surveyors

## Submission
**Hybrid.** Fill the PDF, get notarized, upload to SC LLR e-Service (or mail).

## Files in this folder
- `SC_PE_Comity.pdf` — blank reference copy
- `SC_PE_Comity_PREFILLED.pdf` — ready to print; fill TBD markers + notarize before submission

## Fee
**~$110** initial application fee. Check payable to **"SC LLR - Board of Engineers"**, or pay via portal.

## What to include in the envelope / upload
1. Comity application (notarized signature block)
2. Personal Reference Questionnaire (sent directly by each reference PE)
3. Work Experience Questionnaire (SC requires its own, not satisfied by NCEES Record)
4. $110 check (or portal payment)
5. Copy of Permanent Resident Card (legal presence verification — VLC block on page uses this)

## Notary
**REQUIRED.** The affidavit/signature block on page 5 must be notarized before submission.

## NCEES transmittal
Order NCEES Record transmittal to **South Carolina Board of Registration for Professional Engineers & Surveyors** via your NCEES account.

## Online upload portal
https://eservice.llr.sc.gov/documentsubmission/

## Before you sign
Fill these TBD fields on the prefilled PDF (then notarize):
- Education: BS grad date
- Exam dates
- Reference PE #2, #3 (name + address — SC will mail each one a reference form)
- Fully filled Work Experience Questionnaire rows (SC wants detailed subordinate vs. professional work breakdown per employer)

## Processing
~6-10 weeks; board approves at monthly meetings.
