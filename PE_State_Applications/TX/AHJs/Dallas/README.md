## 1. Overview

Dallas operates a registered **Third-Party Plan Review provider program** through the Department of Sustainable Development & Construction (SDC), Building Inspection Division. Originally launched September 2011 for the Green Building Program (residential + commercial categories); HB 14 (88R, 2023) layered the 15-business-day statewide backstop on top.

The City maintains a published **"Approved Third-Party Plan Review Registered Vendors"** PDF list. Becoming a registered vendor requires:
- Application to SDC
- **Mandatory orientation attendance** (separate orientations for single-family vs commercial)
- Required certifications (ICC/USGBC for green; ICC for general code)

Dallas Service First Bulletin **303** governs the third-party plan review procedural requirements.

## 3. Eligibility

| Discipline | Required Credential |
|---|---|
| Residential Green | ICC Green Building cert OR USGBC LEED AP Residential OR Green Built Texas Verifier; **2 yrs experience**; pass Dallas Green Building Code exam |
| Commercial Green | ICC Commercial Green Construction Professional (GCP) OR USGBC LEED AP Commercial; **2 yrs experience**; pass code exam |
| General Code TPR (HB 14) | TX-licensed PE/RA OR ICC cert per trade |
| Special Inspections | TX PE/RA per IBC Ch. 17 |
