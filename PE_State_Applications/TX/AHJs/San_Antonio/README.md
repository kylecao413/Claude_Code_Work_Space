## 1. Overview

San Antonio's Development Services Department (DSD) operates a formal **Special Inspector Approval program** governed by **Information Bulletin 132** (IB 132) plus a **3rd Party Inspection Option** under **Information Bulletin 133** (IB 133). Both are documented and the application/registration process is well-defined.

**Key advantage for BCC:** Per IB 132, Texas-licensed PEs and Registered Architects are **EXEMPT from submitting qualification certifications** — they only need to provide their license number on the Special Inspection Log Sheet. Once Yue's TX PE issues, San Antonio is the **fastest path to operational status** of all 7 TX AHJs reviewed.

Registration fee is currently **$0.00** (waived "until department deems necessary"). Annual renewal.

IB 133 separately governs the residential 3rd-party inspection option (foundation/slab/pier inspections via 3rd Party Residential Inspection Form).

## 3. Eligibility

| Path | Requirement |
|---|---|
| **Texas-licensed PE/RA** | EXEMPT from cert submittal — provide license # on Log Sheet only |
| Non-PE special inspector | Submit completed application + resume + ICC/IAS/AWS/AISC/ASCE/ANSI/AWC/AISI/AWCI certs (or other nationally-recognized cert), minimum **3 years** verified relevant work experience, reference letter if applicable |
| HB 14 backstop | TX-licensed PE OR ICC cert |

**BCC fit:** Once Yue's TX PE issues, ZERO certification paperwork required for him personally as a Special Inspector — just file license + Log Sheet. ICC MCP path also works in interim.

## 5. Fees

- **Registration:** $0.00 (currently waived)
- **Annual renewal:** $0.00 (currently waived)
- Per-project fees: BCC sets own with applicant; City still charges standard permit fees

## 6. Required Submission Docs

For TX PE/RA path:
- Completed application form (in IB 132 PDF)
- Copy of TX PE/RA license
- Resume
- (Cert submittal exempt)

For non-PE path:
- Completed application
- Resume showing ≥3 yrs relevant experience
- Copies of ICC/IAS/AWS/etc. certifications
- Reference letter (if applicable)

For 3rd Party Residential Inspection Option (IB 133): 3rd Party Residential Inspection Form per project.

## 7. Plan Review Report Format

Use **Special Inspector Required Report letterhead template** (`SIRequiredReport2018.pdf`) from DSD docs portal. Each report submitted on the firm's letterhead with the Special Inspector ID# (issued upon registration approval).

## 8. Submission Process

- Application: Email **dsdspecialinspections@sanantonio.gov** OR mail to **P.O. Box 839966, San Antonio TX 78283-3966** OR in-person
- Per-project: Special Inspection Log Sheet + Required Reports submitted via DSD portal per project
- IB 133 residential: 3rd Party Form per inspection
