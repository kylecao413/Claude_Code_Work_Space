# San Antonio — Plan Review / Special Inspection Report Templates

San Antonio DSD provides a **Special Inspector Required Report Letterhead** template (rev 2018) — used for all Special Inspection submittals once the firm has a Special Inspector ID# (SIID).

Download:
```bash
curl -L -o SanAntonio_SI_Required_Report_Letterhead.pdf "https://docsonline.sanantonio.gov/FileUploads/DSD/SIRequiredReport2018.pdf"
```

Each per-project Required Report submitted on firm letterhead with SIID#. Reports submitted via DSD permit portal per project.

No formal TPR (plan review) template published — IB 132 + IB 133 govern Special Inspections only. HB 14 plan review backstop submittal format is undefined; confirm with dsdspecialinspections@sanantonio.gov before first submission.
