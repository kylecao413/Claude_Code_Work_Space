# Frisco — PDFs to Download

Frisco does NOT publish a formal HB 14 third-party reviewer/inspector application packet. Contractor Registration is fully online via eTRAKiT (no standalone PDF).

Reference portals:
- eTRAKiT Contractor Registration: https://etrakit.friscotexas.gov/etrakit/
- Plans & Permits (Avolve Cloud): https://frisco-tx-us.avolvecloud.com/

ACORD 25 (Certificate of Insurance form) — current version required when registering. ACORD 25 is a standard form, not Frisco-specific; the contractor's insurance broker provides it.

Track quarterly to see if Frisco BIS publishes a formal TPR roster post-HB 14.
