# Austin, TX — Third-Party Plan Review / Inspection (TPR/TPI)

**Verified:** 2026-04-27
**Compiled by:** Claude Code, BCC Business Automation

---

## 1. Overview

Austin's Development Services Department (DSD) does **NOT yet publish a formal third-party plan review (TPR) firms list** post-HB 14. Instead, Austin runs in-house **Expedited Building Plan Review** + **Quick Turn-Around** + **Overtime Plan Review** programs where City staff handle the review, with the applicant paying a premium for accelerated turnaround.

Austin's "DSD Regulatory Changes — 88th Legislative Session" page confirms HB 14 implementation but does not yet enumerate a formal TPR registration portal. Per-project HB 14 backstop submittals (after the 15-business-day statutory deadline) appear to be the practical TPR/TPI path in Austin today.

This is the **softest of the 7 TX AHJs** — most opportunity is in HB 14 backstop work + Special Inspections under IBC Ch. 17.

## 7. Plan Review Report Format

No published Austin TPR template. Generic IBC compliance certification + markup likely accepted.

## 8. Submission Process

- HB 14: Per-project, after AHJ misses 15-day deadline, applicant + BCC submit certified plan review to DSD via email (commercial: ExpeditedReview@austintexas.gov; preliminary: ppr_request@austintexas.gov)
- Special Inspections: per-project per IBC Ch. 17, submitted via DSD permit portal
