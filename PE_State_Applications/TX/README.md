# Texas — TBPELS

## Submission
**Hybrid.** Online portal preferred; PDF-by-mail still accepted.

## Files in this folder
- `TX_PE_Application.pdf` — blank reference copy
- `TX_PE_Application_PREFILLED.pdf` — ready to print (if mailing); or use as answer key for portal (if applying online)

## Fee
**$75** application fee to TBPELS. Plus separate NCEES transmittal ($175 first state / $100 subsequent). Plus **Texas Ethics Exam $40** (online, open-book, after app accepted).

Payment: online via portal **OR** check by mail.

## What to include
With NCEES Record, the following are **waived**:
- Supplementary Experience Records (SERs)
- Transcripts
- Reference statements
- Exam / prior-license verifications

Still required:
1. Texas PE Application form (this PDF, or portal equivalent)
2. **Texas Engineering Professional Conduct and Ethics Exam** (online, after app accepted)
3. **Criminal history fingerprints via IdentoGO** (Board Rule 133.21) — schedule after app submitted
4. Copy of Permanent Resident Card
5. NCEES Record transmittal

## Notary
Not required with NCEES Record path.

## Before submitting
Fill these TBD fields on the prefilled PDF (or type into portal):
- Primary Branch (engineering discipline) — likely **Electrical and Computer** since that's the primary NCEES discipline
- Exam dates (FE, PE Electrical, PE Civil, PE Fire)
- FE / PE exam locations (city, state)
- NCEES Record transmittal date (after you order it)

## Processing
~8 weeks stated average. Ethics exam and fingerprints must clear before license issues.
