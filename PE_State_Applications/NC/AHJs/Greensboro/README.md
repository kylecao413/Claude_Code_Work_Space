## 1. Statutory & Code Basis

- **NCGS Ch. 160D Article 11** + **NCBC Chapter 17**.
- Greensboro enforces NC State Building Code Ch. 17 SI categories with three locally-mandated documents.

## 2. Program Overview

Greensboro requires three documents per SI project:
1. **Statement of Special Inspections** (per NCBC Ch. 17)
2. **Field Technician Qualifications** (lists each inspector + ICC certs)
3. **Final Special Inspection Report Cover Sheet** (closeout document submitted at CO)

No standalone "third-party agency registration" — instead the NC PE/RA accepts responsibility per project. APRIL automated response system schedules routine inspections (336-373-2400).

## 8. Plan Review / Statement / Final Report Format

- **Statement of Special Inspections** — Greensboro-specific PDF (must be sealed by NC PE/RA, listed against Ch. 17 categories).
- **Final Report** — Greensboro requires the **Final Special Inspection Report Cover Sheet** as the closeout doc. NC PE/RA seal required.

## 9. Submission Process

1. Submit Statement + Field Tech Quals with permit application.
2. Conduct field inspections per Ch. 17.
3. At project closeout: submit Final Cover Sheet (sealed) → CO/CC released.
4. Use the GSO BI Scheduler portal or APRIL phone system to schedule field visits.
