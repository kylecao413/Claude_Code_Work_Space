**Last Verified:** 2026-04-27
**Status:** READY (program documented; multiple PDFs identified; phone + address captured; specific PP coordinator email TBD)

## Overview

Miami-Dade County's **Board and Code Administration Division** of the Department of Regulatory and Economic Resources runs the Private Provider program. The Code Coordination Section registers PP Firms, Private Providers, and DARs under §553.791. Webinar + tutorial available. Two separate portals: REP application (DAR) and PF application (Firm).

## Eligibility

Per FL §553.791:
- FL-licensed PE (Ch. 471)
- OR FL Architect (Part 1, Ch. 481)
- OR FL BCA (Part XII, Ch. 468)

## Insurance Requirements

- Statutory minimum: $1M Pro Liab
- Miami-Dade specifically requires Certificate of Insurance verification
- VERIFY whether $2M is required for high-rise / Threshold Buildings (likely yes given Miami-Dade product control regime)
