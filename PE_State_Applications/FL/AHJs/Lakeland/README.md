# City of Lakeland, FL — Private Provider Registration

**Last Verified:** 2026-04-27
**Status:** READY (program documented; email + phone captured; one PDF form found)

## Overview

City of Lakeland (Polk County) operates Private Provider program under FL §553.791. Submission via **ePlan ProjectDox** + **IMS** (Information Management System) portal. Notable restriction: **PP plan review and inspection are linked** — if PP performs plan review, they must also conduct inspections (cannot do plan review only).

## Application URL + Portal

- Program page: https://lakelandgov.net/departments/community-economic-development/building-inspection/private-provider-plan-review-inspection-guidelines/ (verified 2026-04-27)
- **Notice to Building Official PDF**: https://lakelandgov.net/media/21212/ntbo-updated.pdf
- Portals: ePlan ProjectDox (plan review) + IMS Files (general)

## Eligibility

Per FL §553.791: FL-licensed PE / RA / BCA. Special restrictions:
- PP can only review/inspect for FL Building Code compliance
- PP option only if City has not commenced any plan review or inspection activity
- If PP performs plan review, MUST also conduct inspections (can't split)
- Sign permits: PP inspections allowed, but NOT plan review
- Early Start permits not available for PP projects
- Temporary CO not available for PP-permitted projects
