# Charlotte County, FL — Private Provider Registration

**Last Verified:** 2026-04-27
**Status:** READY (program well-documented; full forms list, $0 fee, dedicated email captured)

## Overview

Charlotte County (Punta Gorda / Port Charlotte) operates a clean Private Provider program with **NO registration fee**. Dedicated PrivateProvider@CharlotteCountyFL.gov inbox. One-time firm registration.

## Application URL + Portal

- Program page: https://www.charlottecountyfl.gov/departments/community-development/building-construction/private-providers-for-plan-review-or-inspections.stml (verified 2026-04-27)

## Eligibility

Per FL §553.791:
- FL-licensed Architect (Ch. 481)
- OR FL Engineer (Ch. 471)
- OR FL BCA (Part XII, Ch. 468)

Authorized representatives must hold the same professional licenses.

## Insurance Requirements

Certificates of insurance required during one-time registration. Statutory minimums apply ($1M Pro Liab). Specific County minimums VERIFY but likely $1M.

## Fees

- Initial firm registration: **$0** (no fee)

## Plan Review Report Format

Charlotte County provides their own **Plan Compliance Affidavit** AND a **Plans Review Project Design Data and Required Inspections Form**. The latter is unique to Charlotte County — required when performing plan review.

## Submission Process

- Email registration package to **PrivateProvider@CharlotteCountyFL.gov**
- Per-project: include completed affidavits and Notice form with permit application
- **Inspection reports must be emailed within 2 business days** per FL §553.791(12)
