# Collier County, FL — Private Provider Registration

**Last Verified:** 2026-04-27
**Status:** READY (registration form URL + email + phone confirmed)

## Overview

Collier County (Naples area) requires **one-time Private Provider registration with Contractor Licensing** before commencing any PP work. Processed through the Building Plan Review & Inspection Division.

- Program page: https://www.collier.gov/Business-Resources/Building-Permits-Construction/Permitting-Resources/Private-Provider
