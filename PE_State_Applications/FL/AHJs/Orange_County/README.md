# Orange County, FL — Private Provider Registration

**Last Verified:** 2026-04-27
**Status:** READY (program documented; SOP + contact emails captured)

## Overview

Orange County (Orlando-area, separate from City of Orlando) operates a mature Private Provider program governed by Section 553.791 F.S. All PP inspections are scheduled through the **Fast Track Online Services portal**. Effective Private Provider Manual dated 6/1/2025.
