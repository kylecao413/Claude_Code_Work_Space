# Hillsborough County, FL — Private Provider Registration

**Last Verified:** 2026-04-27
**Status:** READY (HillsGovHub portal + Private Provider Instructions PDF identified)

## Overview

Hillsborough County (Tampa metro outside city limits) handles all Private Provider registration through the **HillsGovHub** online portal under FL §553.791. The county publishes a dedicated Private Provider Instructions PDF.
