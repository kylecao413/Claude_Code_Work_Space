# Pinellas County, FL — Private Provider Registration

**Last Verified:** 2026-04-27
**Status:** NEEDS_OUTREACH (no public PP-specific page; building dept email captured but need to confirm PP process directly)

## Overview

Pinellas County (St. Petersburg / Clearwater area) handles Private Providers under FL §553.791 but does **not publish a dedicated Private Provider page**. The County Building & Development Review Services dept legally accepts all private provider submissions. Process is conducted through the **Accela Citizen Access** portal.

## Application URL + Portal

- Permitting Guide: https://pinellas.gov/permitting-guide/
- **Accela Citizen Access** portal: https://aca-prod.accela.com/PINELLAS/Cap/CapHome.aspx?module=Building&TabName=Home
- Sample permit application packet: https://pinellas.gov/wp-content/uploads/2024/10/sample-permit-application-packet.pdf
- Pinellas County Construction Licensing Board (separate body): https://www.pcclb.com/

Note: Pinellas County has multiple municipal AHJs inside it (St. Petersburg, Clearwater, Largo, etc.). This README covers the **unincorporated county** building department only.
