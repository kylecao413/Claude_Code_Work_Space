## Overview

FL §553.791 governs all private-provider plan review and inspection in Florida. Lee County operates one of the lowest-friction PP programs in the state — per the existing TPI/TPPR research file (§ 2.1), Lee County has **one-time firm registration with no fee**. This makes it BCC's recommended first FL AHJ entry.
