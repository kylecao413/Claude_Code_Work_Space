# City of Tampa, FL — Private Provider Registration

**Last Verified:** 2026-04-27
**Status:** READY (program well-documented; forms hub + email + Accela portal confirmed)

## Overview

City of Tampa runs a robust Private Provider program with **30% fee reduction** on standard permits (15% in FEMA SFHA flood zones) for PP-permitted projects. Submission through the **Accela Citizen Access** portal.

## Application URL + Portal

- Program page: https://www.tampa.gov/construction-services/private-providers (verified 2026-04-27)
- Forms & docs hub: https://www.tampa.gov/document-group/construction-services-forms-and-documents
- **Accela Citizen Access** (portal): https://aca-prod.accela.com/tampa/Default.aspx
- Accela login: https://aca-prod.accela.com/TAMPA/Login.aspx
- Accela account creation guide: https://www.tampa.gov/document/accela-guide-create-accela-account-98261
- Add Contractor License guide: https://www.tampa.gov/document/accela-guide-add-contractor-license-record-99921

## Eligibility

Per FL §553.791: FL PE / RA / BCA. Tampa accepts PP for new construction, additions, alterations, Assembly Occupancies, and Threshold Buildings. **Limited in Special Flood Hazard Areas (City takes lead on plan review and inspections).**

## Insurance Requirements

- Workers' Compensation (per F.S. 440.02(8))
- General Liability (per F.S. 553.791(16))
- Pro Liab — VERIFY exact $1M or $2M minimum
- 5-year tail coverage on claims-made policies (per §553.791(16))

## Fees

- **30% reduction** on standard building permit fees for PP-permitted projects
- **15% reduction** in FEMA SFHA flood zones
- Plan review reductions applied at application acceptance
- Inspection fee reductions applied at permit issuance
- Initial firm registration fee: VERIFY (not posted)

## Plan Review Report Format

City of Tampa requires use of their prescribed forms (Plan Compliance Affidavit + Inspections Report Checklist). Documents uploaded into Accela.

## Submission Process

- Create Accela Citizen Access account
- Upload registration package + insurance to Accela
- Per-project: upload Notice to Building Official + DAR identification + Plan Compliance Affidavit
- Inspections noticed via Accela by **2:00 PM the preceding day**
- Inspection reports uploaded after each inspection
- Email backup / questions: **CSDHelp@TampaGov.net**
- In-person: Development Services Center, 1400 N. Boulevard, Tampa, FL
