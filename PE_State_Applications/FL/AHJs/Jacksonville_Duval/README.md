# Jacksonville / Duval County, FL — Private Provider Registration

**Last Verified:** 2026-04-27
**Status:** NEEDS_OUTREACH (City of Jacksonville does NOT publish a public Private Provider page; Jacksonville Beach has a separate documented PP SOP)

## Overview

Jacksonville and Duval County are unified under one consolidated government ("City of Jacksonville"). The **Building Inspection Division** handles permits but **does not publish a dedicated Private Provider page** as of 2026-04-27 — confirming § 2.1 research note.

**Sub-AHJ note**: **Jacksonville Beach** (separate municipal AHJ within Duval County) DOES publish a Private Provider Program SOP. Jacksonville Beach work requires separate registration with that city.
