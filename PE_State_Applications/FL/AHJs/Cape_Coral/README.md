# City of Cape Coral, FL — Private Provider Registration

**Last Verified:** 2026-04-27
**Status:** READY (program is well-documented; full forms list and emails captured)

## Overview

Cape Coral operates a comprehensive Private Provider program under FL §553.791(16) — note that statute reference §553.791(16) (5 years tail coverage requirement) is explicit on the City page. Cape Coral is geographically inside Lee County but maintains its own building department and PP registration.

## Application URL + Portal

- Program page: https://www.capecoral.gov/department/community_development/customer_service-permitting/private_provider.php (verified 2026-04-27)
- Portal: paper / email submission to ppregistration@capecoral.gov

## Eligibility

Per Cape Coral page:
- FL-licensed PE (Ch. 471) — Yue Cao, FL PE comity in flight
- OR FL-licensed Architect (Ch. 481)
- OR FL Building Code Administrator (Ch. 468 Part XII)
- OR Standard certificate holders (residential additions/alterations ≤1,000 sq ft only)

## Insurance Requirements

- **Statutory minimum** per FL §553.791(18): $1M / $1M Professional Liability
- **Cape Coral specific**: Per §553.791(16), policies must include **5 years of tail coverage** for claims-made policies — confirmed on page
- General Liability + Workers Comp (or WC exemption filing)
- VERIFY: dollar minimums not stated on page; default to FL statutory $1M but procure $2M for marketability

## Plan Review Report Format

Cape Coral provides a **Plan Compliance Affidavit** (template provided — see forms list). Inspection reports use the city's **Inspection Report** template (also provided — see templates list). These are mandatory templates, not optional.

## Submission Process

**Registration:**
- Email PDF package to: **ppregistration@capecoral.gov**
- OR deliver in person to: 1015 Cultural Park Blvd, Cape Coral, FL 33990

**Per-Project Workflow:**
- Notice to Building Official (signed/sealed/notarized) + List of Inspections + Plan Compliance Affidavit (if plan review) + sealed plans submitted with permit
- Inspection Notification (pre-inspection) → email to **PProvider@capecoral.gov**
- Inspection Report (within 2 business days post-inspection) → email PProvider@capecoral.gov
- Certificate of Compliance + final report card at project close-out
- CO request → **CORequest@capecoral.gov**
