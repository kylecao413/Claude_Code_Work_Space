# City of Orlando, FL — Private Provider Registration

**Last Verified:** 2026-04-27
**Status:** READY (program page + forms hub identified; contact email captured)

## Overview

City of Orlando operates a Private Provider program under FL §553.791. Owners using PPs receive **fee reductions** on plan review and inspection. Submission via direct online application.

## Application URL + Portal

- Submit Private Provider Application: https://www.orlando.gov/Building-Development/Permits-Inspections/Other/Submit-your-Private-Provider-Application
- **Private Provider Forms hub**: https://www.orlando.gov/Building-Development/Permits-Inspections/Get-a-Permit/Forms-Documents/Private-Provider-Forms
- General permitting forms: https://www.orlando.gov/Building-Development/Permits-Inspections/Get-a-Permit/Permitting-Forms-Documents
- Permit lookup: https://permitlookup.cityoforlando.net/WebPermits/
